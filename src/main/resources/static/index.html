Metodos

General

  • POST - /api/registro

    Registro de usuarios de tipo cliente

  • GET - /api/usuario/tipousuario

    Para saber el tipo de usuario, a traves del token

  • POST - /api/login

    Inicio de sesion, devuelve un token

Administrador

  • GET - /api/usuarios

    Para ver todos los usuarios que estan registrados en el sitio

  • POST - /api/registro/empleado

    Registro de empleados

  • GET - /api/productos

    Obtener todos los productos

  • GET - /api/productos/{id}

    Obtener producto por id

  • POST - /api/productos/agregar

    Agregar nuevos productos
